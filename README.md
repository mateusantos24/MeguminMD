## 🚀 Instalação rápida

### 1. Instalar dependências

```bash
npm install
```

Se estiver no Windows e a instalação falhar por dependências nativas:

```bash
npm install --force
```

### 2. Configurar ambiente

```bash
cp .env.example .env
```

Edite o `.env` com suas chaves e configurações.

### 3. Iniciar

```bash
npm start
```

Modo desenvolvimento:

```bash
npm run dev
```

### Scripts úteis

```bash
npm run start:bun
npm run dev:bun
npm run lint
npm run lint:fix
npm run patch:baileys
```
